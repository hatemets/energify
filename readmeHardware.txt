//HARDWARE:

Connection with the ZigBee module:
- Set-up Conbee 2, the ZigBee module used, by installing deConz through https://phoscon.de/en/conbee2/install#windows10
- Open the PhosCon App within deConz and connect smart appliances
- Retrieve info on API by plugins>>REST API Plugin

Instillation of BluetoothView:
- Install Nirsoft trough https://www.nirsoft.net/utils/bluetooth_viewer.html

For API connection:
- This is needed to change states of smart appliances in DeCONZ through info gathered on from Bluetooth data in a Python Code
- Follow the steps in the following link to get access to API gateway https://dresden-elektronik.github.io/deconz-rest-doc/getting_started/

Run the python file (readlog.py) to get access to DEMO
